# 📗 **2. CSC 210 – Data Structures**

*File: `CSC_210_Data_Structures.md`*

## Course Description

CSC 210 introduces essential data structures and their implementation using Python or C++. Students explore efficiency, memory management, and trade-offs between structures. Emphasis is placed on building intuition for algorithmic complexity.

## Learning Outcomes

Students completing this course will be able to:

* Implement common data structures (linked lists, trees, heaps, hash tables)
* Analyze time and space complexity using Big-O notation
* Select appropriate data structures for problem-solving
* Understand recursion and its applications
* Build medium-size programs using modular design principles

## Major Topics

* Abstract Data Types (ADTs)
* Arrays vs linked lists
* Stacks, queues, deques
* Trees (BST, AVL, heaps)
* Hash tables and hashing strategies
* Searching and sorting fundamentals
* Recursion and divide-and-conquer

## Assessments

* Biweekly coding assignments
* Lab exercises
* Midterm and final exam

## Expected Weekly Workload

10–12 hours per week (reading, coding, analysis)

## Prerequisites

CSC 123 or equivalent introductory programming experience.
