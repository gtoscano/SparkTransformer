# 📘 **1. CSC 123 – Introduction to Programming**

*File: `CSC_123_Intro_to_Programming.md`*

## Course Description

CSC 123 introduces foundational programming concepts using Python. Students learn variables, data types, decision structures, loops, functions, and simple algorithms. Emphasis is placed on problem decomposition and writing clean, readable code. No prior programming experience is required.

## Learning Outcomes

By the end of the course, students will be able to:

* Understand core programming constructs (variables, loops, conditionals)
* Write, test, and debug Python programs
* Apply problem-solving skills to develop algorithms
* Use functions for code modularity
* Read and interpret basic technical documentation

## Major Topics

* Python basics and environment setup
* Data types and operators
* Control flow (if/else, while, for)
* Lists, strings, basic data structures
* Functions and modules
* Debugging, testing, and best practices

## Assessments

* Weekly programming assignments
* Two midterm exams
* Final cumulative programming project

## Expected Weekly Workload

8–10 hours per week (readings, coding assignments, debugging practice)

## Prerequisites

None.
