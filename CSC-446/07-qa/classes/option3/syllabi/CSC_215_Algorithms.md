# 📙 **3. CSC 215 – Algorithms**

*File: `CSC_215_Algorithms.md`*

## Course Description

CSC 215 provides a theoretical and practical foundation in algorithms. Students learn strategies for designing efficient algorithms, proving correctness, and analyzing complexity.

## Learning Outcomes

Upon completion, students should be able to:

* Apply algorithmic design paradigms (greedy, dynamic programming, backtracking)
* Analyze algorithms using Big-O notation
* Prove correctness and derive recurrence relations
* Compare algorithmic trade-offs and performance

## Major Topics

* Review of mathematical foundations
* Sorting and selection algorithms
* Recurrence relations and the Master Theorem
* Greedy algorithms
* Dynamic programming
* Graph algorithms: BFS, DFS, shortest path
* NP-completeness overview

## Assessments

* Homework problem sets
* Algorithmic coding tasks
* Midterm exam
* Final exam

## Expected Weekly Workload

10–14 hours per week (math-heavy preparation, coding, proofs)

## Prerequisites

CSC 210 (Data Structures) and basic mathematical maturity.
