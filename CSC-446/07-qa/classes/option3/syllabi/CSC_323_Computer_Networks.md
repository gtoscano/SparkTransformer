# 📘 **4. CSC 323 – Introduction to Computer Networks**

*File: `CSC_323_Computer_Networks.md`*

## Course Description

CSC 323 introduces fundamental networking concepts, protocols, and architectures. Students learn how data moves across networks, how protocols coordinate communication, and how to analyze performance.

## Learning Outcomes

Students will be able to:

* Describe the structure of the Internet and OSI/TCP-IP models
* Explain protocols such as HTTP, DNS, TCP, UDP, IP, ARP
* Understand routing, switching, and congestion control
* Use tools like Wireshark to analyze network traffic
* Implement basic socket programs

## Major Topics

* OSI vs TCP/IP layers
* Link-layer framing and error detection
* IP addressing and subnetting
* Transport protocols (TCP reliability, UDP behavior)
* Application protocols (DNS, HTTP)
* Routing algorithms and architectures
* Network security fundamentals

## Assessments

* Packet analysis labs
* Midterm and final exam
* Socket programming assignments

## Expected Weekly Workload

10–12 hours per week

## Prerequisites

CSC 210 and CSC 215 recommended.
